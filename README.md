# Portfolio

Welcome to my portfolio! This project is built using React and Sanity.io, and it serves as a showcase of my skills, projects, and testimonials. Feel free to explore and get in touch if you have any questions or collaboration opportunities.

## Features

- **Skills**: Explore my skill set.
- **Projects**: View my past work and contributions.
- **Testimonials**: Read what others have to say about working with me.
- **Contact**: Reach out to discuss potential collaborations or projects.

## Technologies Used

- React
- Sanity.io
- Tailwind CSS
- Sass CSS
- Framer Motion
